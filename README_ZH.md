# Unity Style Walk Navigation

## 功能简介

为 Blender 3D 视口提供 Unity Editor 风格的第一人称漫游导航，替代原生 Walk 模式。

- 右键长按 + 移动鼠标进入导航，松开退出（带惯性滑行）
- 指数平滑的移动惯性，无论是起步/刹车/转向，手感都自然
- 支持透视、正交、Camera 三种视图模式各自独立的导航逻辑
- 支持更改视角移动坐标系，导航中按 Tab 切换
- 支持触控板（Mac/Windows），双指滑动控制视角
- 全参数保存在插件全局设置中（不随 .blend 文件变化）
- 自动跟随 Blender 界面语言（目前支持中文和英文）

---

## 安装方法

### Blender 4.2+（推荐）

`编辑 > 偏好设置 > 插件 > 从磁盘安装`，选择 zip 文件。

### 旧格式兼容

删除 zip 中的 `blender_manifest.toml`，解压到 `scripts/addons` 目录，重启 Blender 后在插件列表中启用。

## 使用方法

### 鼠标

| 操作                          | 功能                                    |
| ----------------------------- | --------------------------------------- |
| 右键 + 移动鼠标               | 进入第一人称导航                        |
| W / S / A / D / Q / E         | 前进 / 后退 / 左移 / 右移 / 上升 / 下降 |
| Shift                         | 加速冲刺                                |
| Alt（Windows）/ Option（Mac） | 减速移动                                |
| 滚轮上 / 下                   | 调整移动速度                            |
| Tab                           | 切换视角移动的空间坐标                  |
| ESC                           | 强制退出导航                            |

### 触控板

在 `编辑 > 偏好设置 > 键位映射` 中搜索 `Unity Style Walk Navigation (Trackpad)` 并启用。

| 操作                                 | 功能                   |
| ------------------------------------ | ---------------------- |
| 双指滑动                             | 进入导航并控制视角     |
| W / S / A / D / Q / E                | 移动                   |
| Ctrl（Win）/ Option（Mac）+ 上下滑动 | 调整移动速度           |
| Shift                                | 加速                   |
| Alt（Win）/ Command（Mac）           | 减速                   |
| Tab                                  | 切换视角移动的空间坐标 |
| ESC                                  | 立刻退出               |

### 正交视图

正交视图下自动切换为独立的正交导航模式，短按右键弹出菜单，长按或拖动进入导航。

**WASD 键位（默认）**

| 操作            | 功能                     |
| --------------- | ------------------------ |
| 右键 + 移动鼠标 | 旋转视角（保持正交投影） |
| W / S / A / D   | 上 / 下 / 左 / 右平移    |
| Q / E           | 放大 / 缩小              |
| Tab             | 切换坐标                 |

**QEAD 键位（Unity 风格）**

| 操作          | 功能                  |
| ------------- | --------------------- |
| E / Q / A / D | 上 / 下 / 左 / 右平移 |
| W / S         | 放大 / 缩小           |
| Tab           | 切换坐标              |

## 参数说明

> 注意：Blender 的 hover tip（悬停提示）不支持插件翻译 API，所有悬停提示均显示中文，具体参数详见说明表。

### N 面板（View 标签）

| 参数           | 说明                                                         |
| -------------- | ------------------------------------------------------------ |
| 速度           | 移动速度（单位/秒）。滚轮在导航中实时调整并自动保存。        |
| 滚轮步进       | 每次滚轮调整的速度缩放倍率（默认 1.15）。<br />目标速度 = 当前速度 × 或 ÷ 此值。 |
| 鼠标灵敏度     | 鼠标每移动 1 像素对应的视角旋转量（弧度）。                  |
| FOV 旋转补偿   | FOV 对旋转速度的影响程度。<br />基准焦距 50mm，补偿系数 = <br />`lerp(`<br />    `1,`<br />    ` tan(currentFOV ÷ 2) ÷ tan(defaultFOV ÷ 2),`<br />    ` (1-fovT) × strength`<br />`)`<br />其中 `fovT` 为当前 FOV 在 2°~180° 范围内的归一化值。<br />0 = 不补偿，1 = 完全补偿。 |
| 阻力           | 速度阻力系数：`系数 = 1 - e^(-值 × dt)`。<br />越大越灵敏，越小越有漂移感。 |
| 触控板灵敏度   | 触控板双指滑动每移动 1 像素对应的视角旋转量，独立于鼠标灵敏度。 |
| 触控板调速步进 | 修饰键 + 滑动调速时每次触发的缩放倍率，每累计 20 像素触发一次。 |
| 缩放倍率       | 正交视图中，正交比例的缩放速度与移动速度的乘积。             |

#### 偏好设置

**坐标**

| 参数     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 透视坐标 | 透视视图默认坐标空间：世界空间或视角空间，导航中按 Tab 切换。 |
| 正交坐标 | 正交视图默认坐标空间：世界空间或视角空间。                   |

**键位**

| 参数     | 说明                                                    |
| -------- | ------------------------------------------------------- |
| 正交键位 | WASD（默认，Q/E 缩放）或 QEAD（Unity 风格，W/S 缩放）。 |

**速度修饰**

| 参数            | 说明                                               |
| --------------- | -------------------------------------------------- |
| 冲刺倍率        | 按住 Shift 时的临时速度倍率。                      |
| 慢速倍率        | 按住 Alt（Win）或 Command（Mac）时的临时速度倍率。 |
| 最小 / 最大速度 | 滚轮调速的速度上下限。                             |

**其他**

| 参数           | 说明                                                    |
| -------------- | ------------------------------------------------------- |
| 光标样式       | 导航时的光标外观。                                      |
| 边界边距（px） | 鼠标距视口边缘多少像素内触发传送到对侧，设为 0 则禁用。 |
| 停止阈值       | 惯性滑行速度低于此值时视为已停止（单位/秒）。           |
| 最长时间（s）  | 惯性滑行的硬性时间上限。                                |

## 已知问题

### 触控板在 N 面板中误触发导航

在 3D 视口以外的区域（属性面板、大纲视图等）双指滑动不会触发导航，已修复。但侧边栏仍可能误触发，此问题属 Blender API 限制，暂无解决方案。

### Windows 触控板在滑行期间无法被打断

在 Windows 上，惯性滑行过程中双指滑动不会重新进入导航，必须在滑行结束后才能再次触发。可能与 Windows 和 Mac 上 `TRACKPADPAN` 事件的传递机制差异有关。

### Mac 触控板：按键打断惯性

在 Mac 上，按下键盘按键时系统级触控板 momentum 会被打断，导致视角惯性丢失。

### Blender 4.5 + Vulkan 卡顿（Windows）

在 Windows 上使用 Blender 4.5.x 并启用 Vulkan 渲染后端时可能出现明显卡顿，建议切换至 OpenGL 后端或使用 Blender 4.2.x。
